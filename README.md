# 🏛️ The MET Landing Page

This project is a modern, single-page landing site for **The Metropolitan Museum of Art**. It is designed to provide a seamless user experience, showcasing museum hours, current exhibitions, and high-quality imagery of the collection.

The primary focus of this project was to create a pixel-perfect, maintainable, and fully responsive layout using professional web development standards.

## 🌐 Live Preview

Explore the full potential and design of this landing page live on GitHub Pages:
**[View Live Site](https://romanplyta.github.io/layout_landing-page-The-MET/)**

## 🚀 Technology Stack

* **Structure:** `HTML5` – Clean and semantic markup ensures high accessibility and SEO readiness.
* **Styling:** `SASS / SCSS` – Utilized for writing modular, efficient, and well-organized CSS using variables, mixins, and nesting.
* **Architecture:** `BEM Methodology` – Strictly followed the Block Element Modifier approach to ensure class name predictability and style isolation.
* **Responsiveness:** `CSS3 Media Queries` – The design is fully adaptive, providing a flawless experience across mobile, tablet, and desktop devices.

## ✨ Key Features

* **Responsive Layout:** Tailored design for all screen sizes.
* **Modern UI:** High-quality image rendering and clean typography inspired by the museum's aesthetic.
* **BEM-based Codebase:** Scalable and easy-to-read CSS structure.
* **Optimized Performance:** Fast loading times through efficient styling and semantic structure.

## ⚙️ Installation and Setup

To run this project locally and compile the styles, follow these steps:

1.  **Clone the Repository:**
    ```bash
    git clone https://github.com/romanplyta/layout_landing-page-The-MET.git
    ```

2.  **Navigate to the Project Directory:**
    ```bash
    cd layout_landing-page-The-MET
    ```

3.  **Install Dependencies:**
    ```bash
    npm i
    ```

4.  **Launch the Project:**
    ```bash
    npm start
    ```
    *(This command will start a local development server and open the project in your default browser).*
